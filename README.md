
# News From API

**News From API** is a programme which bring the *Latest current affair headlines* from ***newsapi.org*** and print that headlines with voice over in organized manner.

### Using Module:
- import requests
- import json
- import pywin32 [from win32com.client import Dispatch (from pywin32)]

The requests.get method helps to get the API from newsapi.org and,
with the help of json which actually stands for ( java script object notation)
get the code same as there, and at last by using Dispatch and Speak function
of pywin32 Module we get the voice over of that printed headlines or titles as well.
